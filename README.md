# springboot实现微信登录和推送消息

## 需求分析

### 1. 实体类

管理员：id，账号，密码

用户：uId，openID，昵称，是否推送。

文章：aid，标题，内容，浏览数。

用户_文章：uId，aId，浏览时间。

### 2.采用技术

前端：前端采用了thymeleaf模板,使用了Semantic-ui框架。

后端：springboot+mybatis

### 3.项目流程

前提：用户已关注微信公众号，并能从数据库获取到用户的openId。

后台选择用户并向他们分别推送消息，记录他们是否阅读，浏览时间，这篇推文有哪些用户阅读。
